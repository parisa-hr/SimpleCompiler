### Installation

1. Clone the repository:

```bash
git clone https://github.com/parisa-hr/SimpleCompiler.git
```
2. Change into the project directory and create a build folder:
   
```bash
cd SimpleCompiler
mkdir build build/helpers
make
./main
```
